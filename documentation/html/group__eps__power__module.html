enum EPS_mppt_power_module_state

Power module powerstate.

state to control each power module of the satelite

Enumerator
POWER_MODULE_OFF 

power module is turned off - no power from solar panel to batteries

POWER_MODULE_ON 

power module is turned on- Maximum Power Point Tracking for this solar panel
