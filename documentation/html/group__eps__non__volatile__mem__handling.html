Enumeration Type Documentation

enum EPS_deployment_status

Deployment flag status.

This datatype is used to define the status of the umbilical connector.

Enumerator
DEPLOYMENT_NOT 

deployment has not happened but satellite is properly armed

DEPLOYMENT_OK 

deployment has happened but satellite is properly dissarmed

DEPLOYMENT_UNDEFINED 

deployment sequence droped in unresolved situation

DEPLOYMENT_LAST_VALUE 

deployment status non valid.
