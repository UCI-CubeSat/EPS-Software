Enable or disable the switch control mode. 

\begin{DoxyNote}{Note}
E\-N\-A\-B\-L\-E\-: A\-D\-C analog switches closed if the corresponding I/\-O switch is also closed. When using C\-O\-M\-P1, switch control mode must be enabled. 

D\-I\-S\-A\-B\-L\-E\-: A\-D\-C analog switches open or controlled by the A\-D\-C interface. When using the A\-D\-C for acquisition, switch control mode must be disabled. 

C\-O\-M\-P1 comparator and A\-D\-C cannot be used at the same time since they share the A\-D\-C switch matrix. 
\end{DoxyNote}
